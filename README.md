# AI + RPA 标杆客户共创成果展示网站

这是“AI + RPA 标杆客户共创专项”的静态展示网站，基于 Vite + React 构建，并通过 GitHub Pages 自动部署。

线上地址：

https://kali-artist.github.io/kali-sites/

## 内容模块

- 项目总览
- 客户地图
- 四大探索方向
- 七大重点场景池
- 方案库结构
- 三阶段实施节奏
- AI + RPA 共创方法论

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

GitHub Actions 会在 main 分支 push 后自动构建 `dist` 并部署到 GitHub Pages。
